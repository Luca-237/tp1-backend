# API de Gestión de Dispositivos

Esta es una API REST sencilla construida con Node.js y Express para administrar un inventario de dispositivos (computadoras, routers, etc.).

## Estructura del Proyecto

- `index.js`: Servidor principal y definición de rutas.
- `database.js`: Almacenamiento de datos en memoria.
- `middlewares.js`: Funciones de registro de peticiones y validación de datos.
- `package.json`: Definición de dependencias del proyecto.

## Requisitos

- Node.js instalado.
- npm (administrador de paquetes de Node).

## Instalación

1. Clona o descarga este repositorio.
2. Abre una terminal en la carpeta del proyecto.
3. Instala las dependencias:
   ```bash
   npm install
